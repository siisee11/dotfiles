# NOTICE

** clone this repo into root of user directory **

```bash
$ cd
$ git clone

```
## Link vimrc file

```bash
$ ln -s -f dotfiles/vim/.vimrc ~/.vimrc
```

if nvim
```bash
$ ln -s -f dotfiles/vim/init.vim ~/.config/nvim/init.vim
```

## Link tmux.conf file

```bash
$ ln -s -f dotfiles/tmux/.tmux.conf ~/.tmux.conf
```

## Link zshrc file

```bash
$ ln -s -f dotfiles/zsh/.zshrc ~/.zshrc
```
